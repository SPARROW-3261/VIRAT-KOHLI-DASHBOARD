🏏 Virat Kohli Career Statistics Dashboard Analysis
1. Project Title

Virat Kohli Career Statistics Dashboard Analysis

An interactive Power BI dashboard designed to analyze Virat Kohli's cricket career performance across different formats, highlighting key batting statistics, yearly run trends, boundaries, centuries, and overall career achievements.

2. Short Description / Purpose

The Virat Kohli Career Statistics Dashboard provides a comprehensive analysis of one of the world's greatest cricketers. The dashboard enables users to explore batting performance metrics such as total runs, average, strike rate, innings played, boundaries scored, and century records across different cricket formats.

This dashboard helps cricket enthusiasts, sports analysts, and data professionals gain valuable insights into Virat Kohli's career progression and achievements.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Dashboard creation and visualization.
📂 Power Query – Data cleaning and transformation.
🧠 DAX (Data Analysis Expressions) – Calculated measures and KPIs.
🔗 Data Modeling – Relationship creation and data management.
📁 Excel/CSV Dataset – Source data containing Virat Kohli's career statistics.
🎨 Custom Formatting – Interactive dashboard design and KPI cards.
4. Data Source

Source: Publicly available cricket statistics datasets.

The dataset includes:

Match-wise batting records
Year-wise run statistics
Number of innings played
Batting average
Strike rate
Number of 4s and 6s
50s and 100s
Format-wise performance (ODI, Test, T20I)
5. Features / Highlights
Business Problem

Cricket statistics are often spread across multiple websites and scorecards, making it difficult to quickly analyze a player's overall performance and career trends.

Key questions include:

How many runs has Virat Kohli scored?
What is his batting average and strike rate?
How has his performance changed over the years?
How many boundaries and centuries has he scored?
Which years were his most successful?
Goal of the Dashboard

The dashboard aims to:

Provide a complete overview of Virat Kohli's batting career.
Analyze yearly performance trends.
Compare statistics across different cricket formats.
Present important KPIs in a visually appealing manner.
Enable users to interactively filter and explore data.

6. Key Insights
📈 Performance Consistency

Virat Kohli has maintained exceptional consistency throughout his career, reflected in his high batting average of 58.83.

🚀 Aggressive Yet Reliable

A strike rate of 91.67 demonstrates his ability to score quickly while maintaining consistency.

🏏 Boundary Dominance

With 1377 fours and 170 sixes, a significant portion of his runs comes through boundaries.

📊 Strong Career Growth

The yearly run analysis highlights several peak years where Virat Kohli consistently scored over 1000 runs.

🌟 Century Machine

The dashboard showcases his remarkable ability to convert starts into big scores through numerous centuries and half-centuries.
